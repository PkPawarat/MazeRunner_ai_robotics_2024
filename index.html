Welcome to Maze Runner

This project is part of the "Artificial Intelligence (AI) in Robotics" subject at the University of Technology Sydney. The primary objective is to develop an AI capable of executing a specific task using machine learning techniques. In this project, we focused on creating an autonomous vehicle designed to traverse a maze by learning to follow a path using Deep Q-Network (DQN) reinforcement learning while avoiding obstacles. The goal is to simulate a real-world scenario where an autonomous vehicle navigates through hazardous or dangerous areas, making real-time decisions to follow the path and avoid obstacles to reach its destination safely.

Project Details

Problem Statement

The primary problem addressed by this project is the challenge of enabling an autonomous vehicle to navigate through a complex and hazardous environment, such as a maze, by making real-time decisions. The vehicle must learn to follow a designated path while simultaneously avoiding obstacles, simulating real-world scenarios where autonomous systems must operate in unpredictable and potentially dangerous conditions.

Objectives

  • Develop AI by using DQN to train the model of autonomous vehicles.
  • The vehicle must be able to make real-time decisions to follow the derived path while avoiding a collision.
  • Evaluate the AI model’s efficiency, safety, and reliability performance.

Tools Used

  • Programming Languages: Python
  • Framework and Libraries: PyTorch, OpenAI Gym
  • Simulation Tools: PyBullet
  • Version Control: Git, GitHub

Machine Learning technique: DQN

In DQN training, the agent interacts with the environment by selecting actions based on a random or current policy. Replay memory stores every experience—state, action, reward, and subsequent state. The agent randomly samples batches from this memory to train the policy network and optimize it by reducing the loss between the expected and target Q-values. The target network is changed periodically to follow the policy network to ensure stable learning. The agent learns efficient navigation techniques through this cycle of action selection, as well as experience storing, batch sampling, optimization, and network updating.

Methodology

  • Develop the environment (maze) then simulate them in PyBullet.
  • Develop a DQN algorithm to train the model.
  • Develop reward system to train the agent.
  • Evaluate the model’s performance using evaluation metrics.
